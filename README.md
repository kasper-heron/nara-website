# NARA — markedsside

Markedsføringssiden for NARA (naraflow.no). Bygget med Vite, TypeScript, React, shadcn-ui og Tailwind CSS.

## Utvikling

```sh
npm install
npm run dev
```

Server kjører på port 8080.

## Bygg

```sh
npm run build
```

## Deploy

Deployes til Vercel (`vercel.json` i denne mappen).
